# Chatt-gpt
This is the clone of the chatgpt using gemini api

# How to run this chatgpt
## 1.Clone the repository
    '''bash 
        git clone https://github.com/kishore2507ms/Chatt-gpt.git
    '''

## 2. Create the environment
    '''bash
        cd CHATT-GPT
        conda create -n chattgpt python
        conda activate chattgpt
        pip install -r requirements
        python app.py
    '''
